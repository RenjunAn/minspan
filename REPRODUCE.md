# Reproducibility map

Every table/figure in the paper, the data behind it, and the current status of
the chain. Raw run outputs are local-only (see `docs/HISTORY.md`); normalized
result data lives in `results/` and is regenerated by the collectors:

```bash
python scripts/collect_agentdyn.py   # AgentDyn tables + scatter figure data
python scripts/collect_piarena.py    # PIArena tables + per-dataset figure data
```
