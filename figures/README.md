# Figures

`src/fig_*.py` renders the paper figures; shared style in `src/paperstyle.py`.

- `fig_scatter.py` → `security_utility_scatter.pdf`, from `src/agentdyn_scatter.csv`
  (regenerated by `scripts/collect_agentdyn.py`)
- `fig_perdataset.py` → `piarena_per_dataset.pdf`, from `src/piarena_per_dataset.csv`
  (regenerated by `scripts/collect_piarena.py`)
- `fig_latency.py` → `cumulative_latency.pdf`, from `results/defense_ops.csv` and
  `results/local-eval/p3_direct_test.json`. Note: the MinSpan per-call latency is
  currently the eager-mode local measurement; re-measure with torch.compile before
  updating the paper (the paper cites the compiled 2.26 ms median).
- `fig_method.py` → `method_overview.pdf`, the Figure 1 method schematic (draft).

Every figure regenerates from data produced by the collectors in `scripts/`.
